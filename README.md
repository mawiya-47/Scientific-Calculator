## вңЁ Features

### рҹ”ў BASIC Mode
- Trigonometry: `sin`, `cos`, `tan`
- Logarithms: `log` (base 10), `ln` (natural)
- Powers: `xКё`, Square root `вҲҡ`, Absolute value `|x|`
- Factorial `n!`, Percentage `%`
- Constants: `ПҖ` (pi), `e` (Euler)
- Parentheses support `( )`
- Full keyboard support вҢЁпёҸ

### рҹ”¬ SCI Mode
- Inverse trig: `sinвҒ»В№`, `cosвҒ»В№`, `tanвҒ»В№`
- Hyperbolic: `sinh`, `cosh`, `tanh`
- `logвӮӮ`, Cube root `вҲӣ`, `xВІ`, `xВі`
- `1/x` (reciprocal), `Вұ` (negate)
- Scientific notation: `EXP`
- **DEG/RAD toggle** вҖ” ek click mein switch
- **Calculation History** вҖ” pichle 20 calculations

## рҹ§  How It Works

Calculator ek custom **expression parser** use karta hai:

1. User input ek string mein store hota hai
2. Button press pe string mein functions append hote hain (`sin(`, `sqrt(`, etc.)
3. `=` press pe string transform hoti hai вҖ” trig functions DEG/RAD mode ke hisaab se wrap hote hain
4. `Function()` constructor se safely evaluate hota hai controlled scope mein
5. Result display pe render hota hai aur history mein save

**Stats engine** pure JavaScript mein implement hai вҖ” mean, median, mode, standard deviation, geometric mean, harmonic mean sab scratch se.
